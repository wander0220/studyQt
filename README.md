### 시그널과 슬롯
- 시그널과 슬롯은 오브젝트 간의 통신을 위한 매커니즘 
	- GUI에서 이벤트 실행했을 때 지정한 함수를 실행하기 위해서 시그널과 슬롯 사용하면 원하는 함수를 수행할 수 있는 매커니즘 제공.
	- 시그널 : 이벤트가 발생하면 처리
	- 슬롯 : 시그널로부터 받은 이벤트 처리할 함수를 지정해주는 함수
	- 항상 시그널:슬롯 = 1:1로 연결해 사용하는 건 x, n:1, 1:n도 가능.
